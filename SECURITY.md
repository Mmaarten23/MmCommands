# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |

## Reporting a Vulnerability

Vulnerabilities should be reported over at https://discord.gg/bGAnAh99Z5. Either privately message `Mmaarten#1769` or open a support ticket.
They will be evaluated and fixed as soon as possible. The result of this evaluation may or may not be shared with the reporter.
